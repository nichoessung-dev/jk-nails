# JK Nails

Single-file, hand-written website for JK Nails, a neglesalong in Kristiansand, Norway. No frameworks, no build step: all HTML, CSS and vanilla JS live in [`index.html`](index.html).

## Live site

Served with GitHub Pages from the `main` branch root: **https://nichoessung-dev.github.io/jk-nails/**

## Local preview

Open `index.html` in any browser. Nothing to install.

## Structure

- `index.html` — the entire site (brand system, hero, services, about, gallery, reviews, practical info)
- `.github/workflows/claude.yml` — Claude Code GitHub Action (mention `@claude` in an issue or PR to run it)

## Notes

- The gallery cards are CSS/SVG color compositions used as placeholders. Swap them for real photos from [@jk_nails_](https://instagram.com/jk_nails_) when available.
- Booking runs through Setmore: https://jknails.setmore.com/book
