# Project guidelines for JK Nails

This repository is a single, self-contained website. Keep it that way.

## Hard rules

- Everything ships in `index.html`: hand-written HTML, CSS and vanilla JS in one file. No frameworks, no build step, no external JS, no CSS libraries.
- At most two Google Font families, loaded with `display=swap`. No other external assets: watercolor and grain are rendered in code (SVG filters, gradients).
- All visible copy is Norwegian (bokmål). Match the existing warm, direct voice.
- No em dashes anywhere, in copy or comments. Use commas, periods or colons.
- Keep WCAG AA contrast, honor `prefers-reduced-motion`, and never introduce horizontal scroll from 320px up.
